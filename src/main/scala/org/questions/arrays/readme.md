# ArrayMaxSumNonConsecutive
Given an array of numbers, find the highest sum of non-consecutive numbers.
Example :
[1, 2, 3, 4, 5]
 !     !     !
1 + 3 + 5 = 9. Therefore, 9 is the answer.

# TwoSum

# LongestIncreasing
